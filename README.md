Python Projects

## Files

### calculator.py

A basic calculator program that takes two numbers and an operator from the user and performs arithmetic operations (addition, subtraction, multiplication, and division). Includes error handling for invalid inputs and checks for division by zero.

### dice rolling game.py

A simple interactive dice rolling game where players can roll two dice and see the results. The program uses the `random` module to generate random numbers between 1-6 and allows the player to roll multiple times by entering 'y' or exit the game.

### number guessing game.py

A number guessing game with four difficulty levels (Easy, Medium, Hard, and Expert). Players must guess a randomly generated number within a limited number of attempts. Each level has a different range and number of guesses, with feedback indicating if the guess is too high or too low.
